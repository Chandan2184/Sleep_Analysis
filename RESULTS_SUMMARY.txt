================================================================================
SLEEP QUALITY ANALYSIS - COMPUTED RESULTS
================================================================================

To run the full analysis and generate all visualizations and detailed results,
execute the analysis scripts:

  python3 scripts/run_all_analysis.py

or

  python3 run_analysis_simple.py

This will generate:
  - K-means clustering analysis with cluster visualizations
  - Linear regression models with coefficient tables
  - Decision tree analysis with feature importance
  - All plots saved in results/figures/
  - All tables saved in results/tables/

================================================================================
BASIC STATISTICS (From Prepared Dataset)
================================================================================

The prepared dataset contains 5,533 participants with 34 variables covering:
  - Sleep outcomes (duration, quality, sleepiness)
  - Smoking patterns (status, cigarettes per day)
  - Alcohol consumption (status, drinks per day)
  - Demographics (age, gender, income, race)
